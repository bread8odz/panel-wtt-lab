wtt · 统一面板说明

面向轻度运维与日常配置的聚合面板，提供简洁的节点状态、资源管理与快速操作入口。

集中 · 轻量 · 可配置
📋

摘要视图

登入后默认展示关键指标卡片：在线时长、流量概况、最近告警。减少跳转层级，信息密度适中。

适用 · 日常状态总览
⚙️

模块化布局

支持自定义仪表板组件，用户可拖拽重组常用模块。面板基于角色权限，只显示已授权功能。

灵活 · 按需配置
🔐

凭证管理

内置加密存储与一次性令牌机制，支持多账户快速切换。所有传输通过 TLS 加密，不记录明文密钥。

安全 · 本地会话

使用建议

  • 首次登录后建议检查通知频道与阈值配置
  • 每项操作均记录审计日志，可追溯来源 IP
  • 面板支持浅色/深色主题，于个人设置切换
  • 建议定期更新令牌，降低重放风险

场景与说明

快速诊断 在面板内直接 ping／traceroute 目标，无需额外工具。

脚本集成 提供只读 API 端点，可嵌入监控流或内部工具。

面板本身不存储业务数据，所有展示信息来源于对接的服务端拉取。建议通过 wtt 官方文档了解字段释义与刷新策略。

若需直接访问面板控制台，可前往 wtt ，并使用已授权账户登录。
面板是否支持多语言？
当前界面为中文，英文版本正在内部测试；部分模块标题保留英文标识。
令牌过期后如何续期？
在个人安全设置中可生成新令牌，旧令牌即刻失效。建议保留一个备用令牌。
能否隐藏不使用的卡片？
可以。仪表板编辑模式下可隐藏／排序卡片，配置会保存至当前账户。
